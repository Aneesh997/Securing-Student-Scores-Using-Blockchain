# Securing Student Scores Using Blockchain

## Overview

This project focuses on securing student academic records and scores using **Blockchain Technology**. The system provides a decentralized, transparent, and tamper-proof solution for storing and managing student information securely.

Traditional academic record systems rely on centralized databases that are vulnerable to unauthorized access, data tampering, and security breaches. This project solves these issues by implementing **Ethereum Blockchain** and **Smart Contracts** to ensure immutability, transparency, and secure access control for student records.

The project uses **Solidity smart contracts** deployed on the **Ethereum Sepolia Testnet** to securely store student data such as academic scores, attendance, extracurricular activities, and personal information.

## Project Workflow

1. User registers and verifies identity using OTP
2. Smart contract authenticates user role
3. Authorized users add or modify student records
4. Student data is encrypted and stored on blockchain
5. Blockchain generates immutable transaction records
6. Authorized users retrieve and verify records securely
7. Audit trail maintains transparency and integrity
